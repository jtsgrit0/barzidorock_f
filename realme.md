# 🍻 BarZidoROCK - 당신의 밤을 특별하게
홍대, 이태원부터 전국의 모든 펍과 라이브 공연장을 한 곳에서.  
밤의 문화를 즐기는 당신을 위한 올인원 플랫폼
---
## 🌃 앱 소개
BarZidoROCK은 매일 밤 색다른 경험을 찾는 당신을 위해 만들어졌습니다.  
홍대의 활기찬 클럽, 이태원의 감성적인 재즈바, 건대의 아늑한 펍까지...  
지금 당장 근처에서 열리는 라이브 공연을 확인하고 특별한 밤을 시작하세요.
> "단순한 지도 앱을 넘어, 당신의 밤을 풍요롭게 만드는 경험"
## ✨ 핵심 기능
### 🗺️ **지도에서 한눈에 확인**
구글 지도 위에서 내 주변의 모든 공연장과 펍을 실시간으로 확인하세요.  
라이브 공연장은 🔴 네온핑크, 펍은 🔵 시안색 마커로 한눈에 구분할 수 있습니다.
### 📍 **지역별 필터링으로 쉽게 찾기**
- 홍대, 이태원, 강남, 건대, 신촌 등 핫플레이스별 필터
- 라이브 공연장 / 펍 카테고리 분류
- 록, 재즈, 인디, 힙합 등 공연 장르 검색 지원
### 🎤 **오늘의 라이브 공연 일정**
이번 주말 놓칠 수 없는 공연들을 홈에서 바로 확인!  
매진 임박 공연은 특별히 강조해서 보여드려요.
### 💬 **상세 정보와 리뷰**
- 영업시간, 전화번호, 주소 한번에 확인
- 전화걸기 / 길찾기 바로 연결
- 실제 사용자들의 리뷰와 평점
- 내가 찜한 장소 모아보기
## 🎨 앱의 특징
### 🌙 **감성적인 다크 테마**
밤에 사용하기 편안한 다크 모드로 네온사인 같은 시각적 경험을 선사합니다.
### ⚡ **빠르고 부드러운 성능**
Flutter로 개발되어 어떤 기기에서도 부드러운 스크롤과 빠른 로딩을 경험하세요.
### ☁️ **실시간 데이터 동기화**
Firebase로 언제나 최신 공연 일정과 새로운 장소 정보를 받아보실 수 있습니다.
## 🛠 기술 스택
| 기술 | 사용 이유 |
|------|-----------|
| **Flutter** | 크로스플랫폼 개발로 iOS/Android 동시 지원 |
| **Firebase Firestore** | 실시간 데이터베이스로 장소/이벤트 정보 관리 |
| **Google Maps API** | 정확한 지도 서비스와 마커 기능 |
| **Provider** | 효율적인 상태 관리로 앱 성능 최적화 |
| **etc.** | cached_network_image, flutter_rating_bar, url_launcher |
## 🚀 개발자 환경 설정
### 1. Firebase 프로젝트 설정
```bash
# Firebase 콘솔에서 프로젝트 생성 후
flutter pub add firebase_core cloud_firestore
```
- Android: `android/app/google-services.json` 배치
- iOS: `ios/Runner/GoogleService-Info.plist` 배치
### 2. Google Maps API 키 발급
- Google Cloud Console에서 Maps SDK 발급
- `android/app/src/main/AndroidManifest.xml`
- `ios/Runner/Info.plist`에 API 키 추가
### 3. 앱 실행
```bash
flutter pub get
flutter run
```
